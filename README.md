## Why This Exists

Every B2B company spends 30-60% of revenue on sales. Most of that goes to repetitive tasks: researching prospects, writing emails, following up, updating CRMs. AI agents can now handle 80% of this workflow autonomously.

**The problem:** There's no good curation. Tools are scattered across GitHub, Product Hunt, and Twitter threads. This repo fixes that.

## 📖 Resources

### Articles & Guides
- [How to Build an AI SDR](https://www.anthropic.com/research) — Best practices for autonomous sales development reps
- [The AI Sales Playbook](https://docs.composio.dev) — Using tool integrations for sales automation
- [Cold Email Best Practices with AI](https://www.lemlist.com/blog) — Data-driven cold email strategies

### Sales Frameworks (for building agents)
- **BANT** — Budget, Authority, Need, Timeline (qualification)
- **MEDDIC** — Metrics, Economic Buyer, Decision Criteria, Decision Process, Identify Pain, Champion
- **SPIN Selling** — Situation, Problem, Implication, Need-payoff (discovery)
- **Challenger Sale** — Teach, Tailor, Take Control (engagement)

### Key APIs for Building Sales Agents
- **[Apollo.io API](https://apolloio.github.io/apollo-api-docs/)** — Lead database + enrichment
- **[Clay](https://clay.com)** — Waterfall enrichment across 100+ sources
- **[Instantly.ai API](https://instantly.ai)** — Cold email at scale
- **[Lemlist API](https://developer.lemlist.com/)** — Multi-channel outreach
- **[LinkedIn Sales Nav](https://business.linkedin.com/sales-solutions)** — Premium lead data
- **[HubSpot API](https://developers.hubspot.com/)** — CRM integration
- **[Salesforce API](https://developer.salesforce.com/docs/atlas.en-us.api_rest.meta/api_rest/)** — Enterprise CRM

## The Gaps 🔴

These are the massive opportunities — high demand, zero good open-source supply:

| Gap | Impact | Difficulty |
|-----|--------|-----------|
| **Autonomous CRM updater** (reads emails/Slack, updates deals) | 🔥🔥🔥 | Medium |
| **AI deal scoring** (predicts close probability from signals) | 🔥🔥🔥 | Hard |
| **Meeting prep agent** (researches prospect before every call) | 🔥🔥 | Easy |
| **Post-call action agent** (extracts action items, updates CRM, sends follow-ups) | 🔥🔥🔥 | Medium |
| **Competitive intel agent** (monitors competitor pricing/features/hiring) | 🔥🔥 | Medium |
| **Revenue operations agent** (pipeline forecasting, territory planning) | 🔥🔥🔥 | Hard |

**If you're building any of these, submit a PR immediately.**

## Contributing

This repo has a bias toward action. We want tools that **work**, not tools that have nice READMEs.

### Criteria:
- ✅ Open source with a permissive license
- ✅ Has actual working code (not just a README)
- ✅ Solves a real sales workflow problem
- ✅ Maintained (commits in last 3 months)
- ❌ No paid products disguised as open-source
- ❌ No "awesome" lists of links to SaaS products

### How to submit:
1. Fork this repo
2. Add your tool to the appropriate category
3. Include: Name, GitHub link, star badge, one-line description
4. Submit a PR
