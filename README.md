## Why This Exists

Every B2B company spends 30-60% of revenue on sales. Most of that goes to repetitive tasks: researching prospects, writing emails, following up, updating CRMs. AI agents can now handle 80% of this workflow autonomously.

**The problem:** There's no good curation. Tools are scattered across GitHub, Product Hunt, and Twitter threads. This repo fixes that.

### Quick Start

```bash
pip install composio        # or: npm install composio
composio login
composio add hubspot        # OAuth2 — connect your CRM
composio add apollo         # API key — lead enrichment
composio add gmail          # OAuth2 — email outreach
composio add slack          # OAuth2 — notifications
```

Then point any AI agent at the skill file and go.

- **BANT** — Budget, Authority, Need, Timeline (qualification)
- **MEDDIC** — Metrics, Economic Buyer, Decision Criteria, Decision Process, Identify Pain, Champion
- **SPIN Selling** — Situation, Problem, Implication, Need-payoff (discovery)
- **Challenger Sale** — Teach, Tailor, Take Control (engagement)

## Contributing

This repo has a bias toward action. We want tools that **work**, not tools that have nice READMEs.

### Criteria:
- ✅ Open source with a permissive license
- ✅ Has actual working code (not just a README)
- ✅ Solves a real sales workflow problem
- ✅ Maintained (commits in last 3 months)
- ❌ No paid products disguised as open-source
- ❌ No "awesome" lists of links to SaaS products

### How to submit:
1. Fork this repo
2. Add your tool to the appropriate category
3. Include: Name, GitHub link, star badge, one-line description
4. Submit a PR
